# 最短路径贪吃蛇

一个像素风路径谜题小游戏。每个关卡开局会用金币摆出水果图案，玩家操纵贪吃蛇吃完所有金币；只有用最短路径爬满整个图案才算通关。

## 打开方式

直接用浏览器打开：

```text
D:\work\ideaProject\shortest-path-snake\index.html
```

或使用文件 URL：

```text
file:///D:/work/ideaProject/shortest-path-snake/index.html
```

## 操作方式

- 方向键：设置下一步移动方向
- `W` / `A` / `S` / `D`：设置下一步移动方向
- 页面方向按钮：触屏或鼠标操作
- 重开：重新开始当前关卡
- 左右箭头按钮：切换关卡

## 通关规则

- 蛇每吃到 1 枚金币，身体增长 1 格。
- 只能上下左右移动，不能斜走。
- 不能撞到边界。
- 不能碰到自己的身体。
- 贪吃蛇会按固定慢速节奏自动前进。
- 每一步都必须仍然有机会达成最短路径。
- 吃完全部金币时，步数必须刚好等于本关最佳步数。

换句话说，这不是速度型贪吃蛇，而是路径规划谜题：你需要找到一条不重复、不绕路、刚好覆盖整幅金币图案的路线。

## 设计备注

当前 5 个关卡的金币图案是更小的实心像素区域，同时也是一条可验证的最短路线。游戏会记录步数，并在玩家绕路、撞墙、撞到身体或无法继续满足最短路径时立即判定失败。
